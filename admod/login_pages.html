Administration mode

If you see this button in the Navigation bar, you are operating this website in administration mode.

Who can use or access to Administration mode? Normal user can register account to access to Administration mode?

Please note that Administration mode can be used only by Administrators and moderators.

To access Administration mode, you must provide username and password that specially provided to you. Then, you must to complete all of authentication and verification, which designed to identity you and ensure only administrators and moderators can be accessed to Administration mode.

If you try authenticate many times but login attempt failed, your IP address will be recorded and a warning message box will show.
